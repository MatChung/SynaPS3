FEATURES

SynaPS3 has the following features:

  • Support for mounting in a variety of implementations.

  • Universal Payload support, even though some deprecated payloads may not be able to carry out all of SynaPS3's intended features.
    The most supported payload is PL3Dev.
    All of SynaPS3's current features can only be fully implemented on this payload.

  • Functions that may help progress the PS3 homebrew scene further.

  • Functions that aid in the development of Backup Managers and the like.

Functions are described and documented in each of the library source files.

NOTE: SynaPS3 works with PSGroove 1.0, PSGroove 1.1, Hermes v1, Hermes v2, Hermes v3, Hermes v4, PL3, PL3dev. Of all these, PSGroove 1.0 has the least features and PL3dev has the most features. Please be aware of the differences when creating your program.
