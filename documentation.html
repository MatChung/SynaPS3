FEATURES

SynaPS3 has the following features:

  • Support for mounting in a variety of implementations.

  • Universal Payload support, even though some deprecated payloads may not be able to carry out all of SynaPS3's intended features.

  • Functions that may help progress the PS3 homebrew scene further.

  • Functions that aid in the development of Backup Managers and the like.

  • Support for Peek/Poke

FUNCTIONS

Below are documentations on the functions included in the library:

  • void Mount(const char *old_path, const char *new_path)

    Mounts the second argument given to the first argument given. Only supported by PL3 and PL3dev payloads.

    char dev_path[] = '/dev_hdd0/CFW/dev_flash';
    Mount('/dev_flash', dev_path);
  • void MountBDVD(char *path)

    Mounts the argument given to the Bluray drive and /app_home. Supported by all payloads.

    char game_path[] = '/dev_hdd0/GAMES/BLUS00000';
    Mount(game_path);
  • void MountPS2Disc(const char *ps2_path)

    Mounts the argument given to /dev_ps2disc. Can not be used to launch PS2 games (yet). Only supported by PL3 and PL3dev payloads.

    char game_path[] = '/dev_hdd0/PS2GAMES/BLUS00001';
    MountPS2Disc(game_path);
  • void MountPS1Disc(const char *ps2_path)

    Mounts the argument given to /dev_ps1disc. Can not be used to launch PS1 games (yet). Only supported by PL3 and PL3dev payloads.

    char game_path[] = '/dev_hdd0/PS1GAMES/BLUS00002';
    MountPS1Disc(game_path);
  • void RedirectGameData(const char *data_path)

    Mounts the argument given to /dev_hdd0/game. Only supported by PL3 and PL3dev payloads.

    char save_path[] = '/dev_usb003/PS3/game';
    RedirectGameData(data_path);
  • void BootGame(char eboot_path[256], bool highPriority, const char *stack)

    Boots the first argument, with priority given by second (true = 1001, false = 3071), with stack size given by third argument.

    char eboot_path[256]; 
    sprintf(eboot_path, '%s/PS3_GAME/USRDIR/EBOOT.BIN', game_path);
    BootGame(eboot_path, 1, STACK_1M);
  • bool IsBlurayGame()

    Returns true is a valid PS3 game is inserted.

    if(!IsBlurayGame())
    sys_process_exit(1);
  • int GetPayloadType()

    Returns the payload type. 5->PL3dev 4-> PL3 3-> Hermesv3/v4 2-> Hermesv1/v2 or PSGroove 1.1 1-> PSGroove 1.0

    if(GetPayloadType() == 5||GetPayloadType() == 4)
    Mount('/dev_flash', '/dev_hdd0/CFW/dev_flash');
  • bool HasPeekPoke()

    Returns true is the payload has Peek and Poke.

    if(HasPeekPoke())
    pokeq(0xE92296887C0802A6ULL, 0x80000000000505d0ULL);
  • void FixController()

    If the payload has Peek/Poke, activates the controller fix.

    FixController();
  • void pokeq(uint64_t addr, uint64_t val)

    Modifies first argument address in memory to second argument's value.

    pokeq(0xE92296887C0802A6ULL, 0x80000000000505d0ULL);
  • uint64_t peekq(uint64_t addr)

    Returns the value at first argument address.

    peekq(0xE92296887C0802A6ULL);
  • void LoadIOFSModules()

    Loads the IO and FS modules

    LoadIOFSModules();
  • void UnloadIOFSModules()

    Unloads the IO and FS modules

    UnloadIOFSModules();

NOTE: SynaPS3 works with PSGroove 1.0, PSGroove 1.1, Hermes v1, Hermes v2, Hermes v3, Hermes v4, PL3, PL3dev. Of all these, PSGroove 1.0 has the least features and PL3dev has the most features. Please be aware of the differences when creating your program.
