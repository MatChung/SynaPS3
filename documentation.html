FEATURES

SynaPS3 has the following features:

  • Support for mounting in a variety of implementations.

  • Universal Payload support, even though some deprecated payloads may not be able to carry out all of SynaPS3's intended features.
    The most supported payloads are Hermes v3, Hermes v4, and PL3Dev.
    All of SynaPS3's current features can only be fully implemented on these payloads .

  • Functions that may help progress the PS3 homebrew scene further.

  • Functions that aid in the development of Backup Managers and the like.

  • Universal syscall36.

  • syscall35 for Hermes v3 & Hermes v4 payloads.

FUNCTIONS

Below are documentations on the functions included in the library:

  • uint32_t syscall35(const char *old_path, const char *new_path)
    Supported payloads:PL3, PL3Dev
    Please use Mount instead of syscall35. This is only included to increase compatability with existing homebrew.
    NOTE: Hermesv3/Hermesv4 Payloads can only mount 1 directory at a time.

    Mounts the second argument given to the first argument given. This will return FAKE if ran on Hermes v3/Hermes v4.

    char dev_path[] = '/dev_hdd0/CFW/dev_flash';
    Mount('/dev_flash', dev_path);
  • void syscall36(char *game_path)
    Supported payloads: All
    Please use MountBDVD instead of syscall36. This is only included to increase compatability with existing homebrew.

    Mounts the argument given to the Bluray drive and /app_home. This will return FAKE if ran on PL3 or PL3Dev.

    char game_path[] = '/dev_hdd0/GAMES/BLUS00000';
    Mount(game_path);
  • void pokeq(uint64_t addr, uint64_t val)
    Supported payloads: PSGroove1.1/Hermes v1/Hermes v2, Hermes v3/Hermes v4, PL3, PL3Dev

    Modifies first argument address in memory to second argument's value.

    pokeq(0x800000000004ca38ULL, 0x4BFFFFD440990024ULL);
  • uint64_t peekq(uint64_t addr)
    Supported payloads: PSGroove1.1/Hermes v1/Hermes v2, Hermes v3/Hermes v4, PL3, PL3Dev

    Returns the value at first argument address.

    peekq(0xE92296887C0802A6ULL);
  • int ReadFirmwareVersion(char * FirmwareVersion)
    Supported payloads: All

    Writes firmware version in the form of '00.0000' to FirmwareVersion.

    ReadFirmwareVersion(FirmwareVersion)
    if (strcmp(FirmwareVersion, "03.0100")==0){
    	pokeq(0x800000000004ca38ULL, 0x4BFFFFD440990024ULL);
    	pokeq(0x80000000000547fcULL, 0x6000000038de0007ULL);
    	pokeq(0x800000000005487cULL, 0x480000a02f840004ULL);
    }
  • void Firmware342Fix()
    Supported payloads: PSGroove1.1/Hermes v1/Hermes v2, Hermes v3/Hermes v4, PL3Dev

    Returns the value at first argument address.

    Firmware342Fix();
  • bool IsBlurayGame()
    Supported payloads: All

    Returns true is a valid PS3 game is inserted.

    if(!IsBlurayGame())
    	sys_process_exit(1);
  • void BootGame(char eboot_path[256], bool highPriority, const char *stack)
    Supported payloads: All

    Boots the first argument, with priority given by second (true = 1001, false = 3071), with stack size given by third argument.

    char eboot_path[256]; 
    sprintf(eboot_path, '%s/PS3_GAME/USRDIR/EBOOT.BIN', game_path);
    BootGame(eboot_path, 1, STACK_1M);
  • void FixController()
    Supported payloads: PSGroove1.1/Hermes v1/Hermes v2, Hermes v3/Hermes v4, PL3Dev
    NOTE: PSGroove1.1/Hermesv1/Hermesv2 and PL3Dev Payloads only support this on 3.41.

    Activates the controller fix. Limited to 3.41. Remember that PL3/PL3Dev autopatches this.

    FixController();
  • uint32_t Mount(const char *old_path, const char *new_path)
    Supported payloads: PL3, PL3Dev
    NOTE: Hermesv3/Hermesv4 Payloads can only mount 1 directory at a time.

    Mounts the second argument given to the first argument given. This will return FAKE if ran on Hermes v3/Hermes v4.

    char dev_path[] = '/dev_hdd0/CFW/dev_flash';
    Mount('/dev_flash', dev_path);
  • void MountBDVD(char *game_path)
    Supported payloads: All

    Mounts the argument given to the Bluray drive and /app_home.

    char game_path[] = '/dev_hdd0/GAMES/BLUS00000';
    Mount(game_path);
  • void LoadIOFSModules()
    Supported payloads: All

    Loads the IO and FS modules

    LoadIOFSModules();
  • void UnloadIOFSModules()
    Supported payloads: All

    Unloads the IO and FS modules

    UnloadIOFSModules();
  • void Exit()
    Supported payloads: All

    Exit the program.

    Exit();

NOTE: SynaPS3 works with PSGroove 1.0, PSGroove 1.1, Hermes v1, Hermes v2, Hermes v3, Hermes v4, PL3, PL3dev. Of all these, PSGroove 1.0 has the least features and PL3dev has the most features. Please be aware of the differences when creating your program.
